# Button

按钮组件

### 何时使用
标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

### 高级用法
<Playground>
<Button ref={btn => (this.btn = btn)} loading="加载中..." 
    onClick={ () => {
      setTimeout(() => {
        this.btn.enabled();
      }, 1500);
    }}>点击出现加载</Button>
    <br/><br/>
    <Button 
        onClick={() => {
          return new Promise((resolve, reject) => {
             setTimeout(() => {
                resolve();
             }, 1500);
          });
        }}>autoDisabled & Promise 的演示</Button>
       <br/><br/>
       <Button 
            loading="加载中..." 
            onClick={() => {
              return new Promise((resolve, reject) => {
                 setTimeout(() => {
                    resolve();
                 }, 1500);
              });
          }}>autoDisabled & Promise & loading 的演示</Button>
</Playground>
