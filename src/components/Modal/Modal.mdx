# Modal

模态框

## Properties 
  空
 
## 函数
  0.  **通用参数: Modal** \[confirm | alert | dialog](options)
  >     以下三个函数都可以带有 title, content, options 字段
      options.title?: React.ReactNode | string // 头
      options.content?: React.ReactNode | string // 内容
      options.options: { // 其他参数
        className?: string // 整体className
        titleCN?: string // 头className
        contentCN?: string // 内容 className
        animateTime?: number // 退出动画时间
        animateName?: string // 自定义动画 className
      } 
      
  0.  **通用参数: Button** 
  >     有Button字段需要传递时, 参考该 Interface
      color?: string // 按钮文字颜色
      backColor?: string // 按钮背景颜色
      content?: React.ReactNode | string // 按钮内容
      className?: string // 按钮特定的 className
      onClick?: function // 按钮回调
      autoClose?: boolean // 点击按钮后是否需要自动关闭?

  1.  **confirm**(options)
  >     弹出 confirm modal
      options: {} (继承 通用参数: Modal 三个字段)
      options.cancelBtn: (继承 通用参数: Button 四个字段)
      options.sureBtn: (继承 通用参数: Button 四个字段)
  
  2.  **alert**(options)
  >     弹出 alert modal
      options: {} (继承 通用参数: Modal 三个字段)
      options.sureBtn: (继承 通用参数: Button 四个字段)
      

  3.  **dialog**(options)
  >     弹出自定义 modal
      options: {} (继承 通用参数: Modal 三个字段)
      options.buttons: [btn, btn, btn...] (是一个数组) (btn 继承 通用参数: Button 四个字段)
  
  4.  **hide**()
  >     隐藏最后一个 modal
  
  5.  **hideAll**()
  >     隐藏所有 modal
  
  6.  **destroy**()
  >     销毁该组件(将不能再被唤起)
