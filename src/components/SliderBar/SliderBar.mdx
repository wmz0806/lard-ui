# SliderBar

列表某一行左右滑动时按钮

## Properties

<PropsTable of={SliderBar} />

## 函数

  1.  **reset**()
  >     重置该组件

  2.  **popButton**(btnOption)
  >     在已经完全弹出的Button上再次弹出弹层
      btnOption: {content: string || React.ReactNode, cb: function}


## Basic usage

<Playground>
  <div style={{width: '100%', height: '40px'}}>
    <SliderBar
      ref={r => (window._btnRef = r)}
      leftBtn={[
        {
          content: '再次弹出蒙层', 
          width: '20%', 
          'backColor': 'red', 
          cb: () => window._btnRef.popButton({content: 123, cb: () => {console.log('弹出层1')}})
        },
        {content: <span>左侧按钮2</span>, width: '10%', 'backColor': 'green', cb: () => console.log('点击2')},
        {content: <span>左侧按钮3</span>, width: '30%', 'backColor': 'yellow', cb: () => console.log('点击3')},
      ]}
      rightBtn={[
        {
          content: '右侧按钮1',
          width: '20%',
          'backColor': 'yelow',
          cb: () => window._btnRef.popButton(
            {
              content: <div style={{height: '100%', backgroundColor: 'green'}}>pop button</div>, 
              cb: () => {console.log('弹出层2')
            }
          })
         },
        {content: '右侧按钮2', width: '30%', 'backColor': 'gray', cb: () => console.log('点击2')},
        {content: '右侧按钮3', width: '15%', 'backColor': 'red', cb: () => console.log('点击3')},
      ]}
    >
      <div style={{textAlign: 'center', background: 'white', height: '100%'}}>
        I am children!
      </div>
    </SliderBar>
  </div>
  
</Playground>
