# TextField
输入组(input 格式下, 下方的错误提示为0.44rem的高度, 使用的position: absolute布局)

## Properties
  <PropsTable of={Props} />

#### headerConfig

#### ref返回实例中调用的函数

  1.  **getRef**()
  >     获取组件的输入节点 (input, textarea)
  2.  **validate**(isShowErr: boolean = true)
  >     对组件发起验证(需要传入了validate规则), 参数为是否显示错误提示
  3.  **getValue**()
  >     获取组件的当前值
  4.  **setValue**(v: string)
  >     设定组件的当前值
  5.  **setDisabled**(isDisabled: boolean)
  >     设置组件当前是否禁用|启用

 <div>

    needClearIcon={true}
    needEncrypt={false}
    validateReg={/^xi/}
    isBlurValidate={true}

    validateMsg={'输入 xi'}

    type={'text'}

    onBlur={(e, v) => console.log('onBlur')}
    onFocus={(e, v) => console.log('onFocus')}

    className={'custom'}
    //isDisabled={false}
  />
 </div>
 
 <div style={{height: '1rem'}}/>
 
 <div
   onClick={() => console.log(window._inputRef.validate())}
 >点我发起验证</div>

    onClick={() => console.log(window._inputRef.getRef())}
